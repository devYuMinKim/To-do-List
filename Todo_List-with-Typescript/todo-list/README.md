# TypeScript로 Todo List 구현

### 구현 기능
- 엔터(or 우측 아이콘)을 통해 할일 추가
- 체크 아이콘 클릭시 완료 처리
- hover 했을 때 생기는 마이너스(-) 버튼 클릭시 제거 처리
- 전체 완료, 전체 삭제 기능 추가

### 메모
- TypeScript로 리액트 파일 생성: npx create-react-app todo-list --template typescript
- useState가 아닌 useReducer을 통해 구현
